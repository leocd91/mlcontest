1. Place the training data file "train.json" in the same directory as the "Final_Submission_Leo_v1.ipynb" file.
2. Open the "Final_Submission_Leo_v1.ipynb" file.
3. Change the "test_file" parameter in line 3 (Part 0 Getting Ready) to the holdout test data file (by default, it is 'test_traces_250.json').
4. Run all the cells in the notebook.
5. The final submission file "submission.json" will be generated.

Optional:
To generate solution pictures, create a folder named "outputpic", uncomment line 37 (Part 4 Prediction & Submission - Reconstruction) and run it.
